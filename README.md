## ⚠️ Важное примечание
Этот проект был создан в 2023-2024 году как дипломная работа в колледже.

Сейчас я изучаю современный стек .NET 8 и понимаю, что этот код содержит устаревшие подходы (отсутствие MVC, прямые SQL-запросы). 
Мои текущие проекты с использованием ASP.NET Core, EF Core и Docker можно найти в репозитории [AutoService](<https://github.com/ZxcPap14/AutoService>).
